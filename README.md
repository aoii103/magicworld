### 模块准备
这里推荐使用[Anaconda](https://www.anaconda.com/download/)集成环境，自带各类常用第三方模块，更高效率开发。
 首先我们通过如下命令安装所需第三方模块
 
```
pip install -r ./requirements.txt
```

### 运行

```
vi conf.py # 填入各项配置
python server.py
```
